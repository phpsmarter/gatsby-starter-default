

# `总体介绍`

随着计算机技术的发展, 机器学习(ML),人工智能(AI)和数据科学(DS)的应用日趋广泛,社会的每个角落现在都能看到这些技术的应用,作为现代社会的成员,了解和学习这些技术是非常必要的. 

数学方法是MADS的基础和核心,从科学家构建最简单的神经网络开始, 依托的工具就是数学.

在有关MADS的学习中涉及的概念非常多, 单凭一本书,或者一个教学视频无法完成任务. 

由于MADS中还涉及计算机编程内容,所以任务就更复杂, 既要理解概念,又要理解具体的程序实现方法. 

替代方法: 通过交互式编程方法来学习MADS的概念.

> 在这个教程里我们的总体方法是使用Julia语言的软件包Pluto Note 来实现概念的交互式学习

## `1.Julia 和Pluto Note介绍`

Julia是一门新的编程语言, 专门用于科学计算, 也就是专长为涉及数学相关的编程,涉及到数学计算的方方面面,很多大型的机构都在构建复杂的软件包,所以学习的潜在价值非常大, 具体内容后面会讲到. 

Pluto 是Julia语言的一个软件包, 借鉴了JavaScript语言 ObservableHQ 软件包的思路.  

$$Pluto\xrightarrow[data\rightharpoondown ]{\leftharpoonup results}Julia$$

Pluto 基本结构有两部分, 一是和使用者交互的部分,就是你现在看到的内容,基于HTML和Javascript(JS)技术,具体来讲是子集 [**Preact**](https://www.preactjs.com.cn/),学习和编程就在这部分完成. 
但是我们看到的编码部分并不是由Pluto来完成, 具体的程序执行部分由你电脑上安装的Julia程序以及它的软件包来完成. 
Pluto会把笔记内的编码发送到Julia, Julia程序执行完以后把结果(无论是正确还是错误的)发送回Pluto. Pluto 在笔记内渲染出结果


交互式方面的内容我们这里不展开讲, 当你打开一个笔记的时候, 你就会知道这是什么意思. 交互式的部分有两个程度的, 基本交互通过PlutoUI软件包提供的多种工具来改变程序中的变量赋值. 复杂的交互,你可以直接修改代码. 


本教程所有的内容都是通过 PlutNote来实现的,既是编程又是数学

## `2.MADS 中的数学`
  为什么数学重要, 用函数的观点解释, 我们可以把物理世界大大小小的各种属性映射(mapping)到数学集合里, 以集合里数字的形式就可以对比属性的差异, 目前来看这是可行的办法. 本教程里会反复的灌输函数映射的思想, 我个人观点看, 数学思维就是映射思维. 
### 主要分支
  #### 微积分
  广义的还包括:数论, 实分析和泛函分析. 基本出发点:连续性.  这些内容经过多年的发展, 已经积累了非常多的结论, 深究每一个具体的数学概念,都可以找到更为抽象和一般化的理论研究. 这里不准备介绍所有的内容. 但是结合函数映射的观点介绍一些思路.
  #### 线性代数
  在这里线性代数的目的还是研究映射, 不过是线性映射,  我们之所以考虑线性映射, 根本原因是线性关系我们可以理解, 通过微积分的连续性, 非线性问题在局部会简化为线性问题. 线性代数在数据科学里无处不在, 我们始终都在处理向量和矩阵.  本教程里所有的可视化内容都是在操作向量和矩阵. 
 ####  概率和统计
  我要说一个很悲观的观点, 很多学习者历经多次统计学习, 任然掌握不了统计学的基本思路,包括我在内. 在本教程里我只想设定一个学习目标: **样本标准误**概念的理解. 在未来的某一天当你突然理解了什么是**样本标准误**的时候, 你可以大步的向数据科学的各个领域迈进.   我们会反复的列举**总体**和**样本** 关系的示例 . 
  统计里还涉及到贝叶斯统计, 如果你没有理解**总体**和**样本**的概念,贝叶斯统计就更难理解. 简单来说贝叶斯统计时根据实验中获取的信息不断的切换研究的总体. 这个概念有点复杂. 需要慢慢理解.贝叶斯统计理论更符合我们人类认知的过程,所以也非常重要
 
#### 数学模型和微分方程
  大众现在对机器学习,人工智能方面的认识都基本基于计算机视觉方面: 指纹识别, 人脸识别等等. MADS的应用不止这一点, MADS加快了各种数学模型的构建, 例如近期比较热门的技术 PINN(物理信息神经网络), 人工智能方面的进展加速了数学模型的构建, 为解决实际问题提供更为快速和精确的方法. 所以我们会强调数学建模方面的内容,包括微分方程的内容

## 4.MADS的数据可视化技术
