## 准备工作
后面的学习中需要使用一些数据表格, 由于 Pluto 是基于浏览器的软件软件包. 所需资源最好是从服务器加载
这里就以 `node.js` 的 软件包 `http-server` 为例讲解一下
### 1. node.js 安装

 [安装步骤参见这里](https://www.runoob.com/nodejs/nodejs-install-setup.html)
### 2. http-server安装
   `node.js` 自带一个软件包管理工具`npm`,`http-server`是一个非常简单的 web 服务器软件包,可以在
    任何目录启动 web 服务
    通过一下命令安装`http-server`

  ```bash
     npm i -g http-server
  ```
  `-g` 的意思是全局安装, 因为有这个参数, 他就可以在任何目录下使用

  在下面链接下载 苹果公司的股票交易
  [苹果公司股票信息](https://github.com/plotly/datasets/blob/master/finance-charts-apple.csv)
  因为有时候直接从这里加载 csv,可能会出问题
    
  在下载的有`finance-charts-apple.csv` 的目录下启动`http-server`
  ```bash
     http-server
  ```
