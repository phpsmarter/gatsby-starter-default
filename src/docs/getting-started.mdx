## 1. 软件包安装提示
> 教程使用 Julia 1.7.x 版本 <br />

- 下载请参考这个网页 [`Julia安装教程`](https://cn.julialang.org/downloads/)
- 国内使用需要设置镜像,请参考这个网页 [`Julia 镜像设置`](https://cn.julialang.org/downloads/)

>最好使用永久设置的方法. 后面使用 Julia软件包,最好通过Pkg 的形式安装, 这样安装是访问国内镜像资源.
>但是有时有些依赖包仍然是时通过Github 仓库来实现的,需要耐心等候. 
>

>安装完之后在终端输入`Julia`.如果现实如下图形就表示成功

可以在你想存储 julia 笔记的路径下运行 julia. 
在 MacOS系统下通过一下命令
```bash

>cd /Users/lunarcheung/Public/github/plutonote/M4MADS/
```
就可以进入存储笔记的文件夹, 然后运行 julia软件

```bash
  julia>pwd()
```
可以查看现在在什么目录下工作. 

这应在下面步骤打开的 Pluto http server 的时候是从进入的文件处打开的.如果对路径问题不熟,可以找比较了解的人咨询一下
这个问题后面总是会遇到,所以早一点了解有好处.

## 3. 安装Pluto 软件包

```bash
julia> ]

(@v1..7) pkg>

(@v1.7) pkg> add Pluto
```
